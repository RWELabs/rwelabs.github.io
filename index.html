What is RWE Labs?

RWE Labs is a subsidiary of Ryan Walpole Enterprises that focuses on specialised software releases that don’t fit within the normal constraints of software engineering at Ryan Walpole Enterprises.

Whether it’s a passion project by an RWE employee, an unsupported modification or add-on to existing software, a patch for legacy software or even just open-sourced software – RWE Labs is where you’ll find it.

Popular Projects

Since RWE Labs is made up of largely hobby projects, our most popular projects can be wildly different from one another. From Minecraft Datapacks to Game Mod Managers and beyond. See our most popular repositories below:
Stardew Valley Mod Manager
and Modded Framework
The Stardew Valley Mod Manager is a powerful tool that is designed to be used alongside SMAPI to help you install and manage mods, automatically install modpacks, manage your game saves and keep your mods up to date.

Much alike SMAPI, The Stardew Valley Mod Manager does not edit or change any of the core game files and is installed as a completely separate executable to both the main game and the SMAPI launcher. This powerful new application allows you to do a variety of different operations pertaining to mod management and game save management.
RyTek AFP
Application Framework Patch
The RyTek3 Application Framework reached end of life support in January of 2017, however many applications were built on this framework and have not been updated since the end of support.

In 2020, apps on Windows 10 running the RyTek3 framework stopped working due to an issue caused by updates made to Windows. This application framework patch is designed to fix that issue and allow those broken applications to still function.

This patch should be used if the user is having issues on modern versions of Windows with software built with RyTek.
