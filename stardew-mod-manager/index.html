Stardew Valley Mod Manager

The Stardew Valley Mod Manager and Modded Framework is an extension to SMAPI, the open-source modding framework API for Stardew Valley that lets players play the game with mods. Much alike SMAPI, The Stardew Valley Mod Manager does not edit or change any of the core game files and is installed as a completely separate executable to both the main game and the SMAPI launcher.

This powerful new application allows you to do a variety of different operations pertaining to mod management and game save management.

Manage Mods

Install Mods and Entire Modpacks
With the Stardew Valley Mod Manager, you can Install Mods and Entire Modpacks with just a couple of clicks. Simply download your Mod from NexusMods (or elsewhere) as a ZIP file, click "Install Mods" and "Install Mod (.zip)" to install a mod. Or alternatively click "Install a modpack" to install an entire set of mods autonomously using a .svmp file, that developers can create to distribute multiple mods.
> How to make a Stardew Valley Modpack
Enable and Disable Mods
With the Stardew Valley Mod Manager, you can have mods enabled (active in game) or disabled (not active in game). The benefit of this is that you don't have to completely uninstall your mods to disable them from your game. Now you can have multiple different game saves with different mods for each save active. Enabling and Disabling a mod is as easy as selecting it and clicking "Enable" or "Disable"
Check for Mod Updates
With the Stardew Valley Mod Manager, you can check compatible* mods for available updates and then update them if updates are available with the Mod Update Check (Beta) added in Version 220304.
> Ensuring my mod is compatible with the Mod Update Check utility
Creating and using Mod Presets
If you run different multiple modded save files, you can use the Stardew Valley Mod Manager to create presets that contain all relevant mod. To create a mod preset, simply add all mods you wish to use to the Enabled mods list, then click "Save Preset". Once you've given it a name, simply click "Load Preset", select your Preset and open it. All of the mods listed in your preset will become Enabled and all other Mods already in the enabled list will be automatically disabled. If you've removed a mod that was referenced in the preset, it will not be added to the enabled list.

Game Save Management

Backup Farms
By default, SMAPI comes with a farm backup utility that runs every time you play Stardew Valley. This backs up all of your saves. However, they are rotational and will only keep the most recently 5 snapshots. The Game Save Management tab allows you to access the SMAPI backups with the click of a button, as well as create a manual backup of your chosen game save with just two clicks.

Backups can be made on any of the farms listed in the "My Farms" list by selecting the farm and clicking "Backup Selected Farm"
Delete and Manage Saves
From the Game Save Management tab, you can open your game saves folder with the click of a button to view all of your game saves. You can also select a game save from the list and click "Delete Farm" to remove the save entirely. This can be useful for any corrupted game saves that are crashing your game on launch.
