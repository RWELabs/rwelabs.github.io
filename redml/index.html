      RedM Launcher

      The RedM Launcher is a custom game launcher for RedM, a Red Dead Redemption server and modding platform.
      The RedM Launcher automatically checks and clears cache files before launching the game. Previously users would be encouraged to do this manually or to create a BAT script shortcut. These solutions are intimidating to new users and can result in errors if the user does not correctly set up their scripts.

      How it works

      When you launch RedM Launcher, the application will automatically scan your AppData folder, ensuring that RedM is installed correctly and then ensuring that your RedM Application Data folder is clear of any cache files.

      Not clearing your cache can result in in-game bugs and glitches that can often be frustrating and hard to diagnose. Many servers will encourage or instruct users to clear their cache files - as they redownload fresh when the user joins the server.
